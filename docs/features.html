Features

Core capabilities of Deyad, from autonomous agents to full local development workflow.

AI Agent

Autonomous planning, code generation, error auto-fix, and task queue with up to 30 iterations per request.

Editor

Monaco editor with multiple language support, FIM completion, and powerful multi-cursor controls.

Database

SQLite with Prisma integration and in-app browser; no database server required.

Deploy

7 deployment targets including Vercel, Netlify, Railway, Fly.io, Surge, VPS, and desktop packager.

Plugin ecosystem

Add tools, agents, and themes via plugins. Plugins are auto-discovered in user data or registry.

Git

Auto-commit, diff previews, version history, and safe rollbacks.
