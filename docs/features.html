AI Agent

Autonomous agent with 18 tools

The agent reads your codebase, plans its approach, writes and edits files, runs shell commands, detects errors from dev server logs, and self-corrects — up to 30 iterations per request.

  • 18 tools: read, write, edit, search, run commands, git, database
  • Error auto-recovery: watches dev server, up to 3 fix attempts
  • Planning mode: generates a plan for your approval first
  • RAG retrieval via Ollama embeddings + TF-IDF fallback
  • Vision: paste a screenshot, get working UI code
  • Task queue for batch processing

IDE

Full-featured editor and workspace

Monaco (the VS Code engine) with syntax highlighting for 15+ languages, fill-in-the-middle code completion, a real terminal, and live preview — all in one window.

  • Monaco editor with multi-cursor, find & replace, minimap
  • FIM autocomplete with any compatible Ollama model
  • Full xterm.js PTY terminal
  • Embedded Vite dev server with live preview
  • Package manager, .env editor, project search
  • Command palette (Ctrl+K)

Database

SQLite, zero config, built-in browser

No Docker, no server, no configuration. Your database is just a file. Prisma ORM handles migrations, and the built-in browser lets you inspect data in real time.

  • SQLite: instant, offline, zero overhead
  • Prisma ORM for type-safe schema management
  • Browse tables, view rows, inspect schema in-app
  • Agent queries live database structure while coding

Version Control

Git-first, auto everything

Every AI generation is automatically committed. Browse a visual timeline, preview diffs, and restore any version with one click.

  • Auto-commit every agent iteration
  • GitHub / GitLab push, pull, and remote management
  • Create, switch, list branches from UI or chat
  • Visual commit timeline with one-click restore
  • Diff preview before accepting changes

Deploy

Ship to 7 targets from one modal

Deploy to Vercel, Netlify, Railway, Fly.io, Surge, any VPS via SSH, or package as a desktop app with Electron Builder.

  • Vercel & Netlify: OAuth or CLI, one-click
  • Railway & Fly.io: full-stack with database
  • VPS: SSH + rsync, optional nginx + SSL
  • Desktop: AppImage / .exe / .dmg
  • Mobile preview via Capacitor
