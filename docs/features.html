AI Agent

Autonomous agent with 18 tools

The agent reads your codebase, plans its approach, writes and edits files, runs shell commands, detects errors from dev server logs, and self-corrects — up to 30 iterations per request.

  • 18 tools: read, write, edit, search, run commands, git operations, database queries
  • Error auto-recovery: watches dev server, auto-sends up to 3 fix attempts
  • Planning mode: generates a plan for your approval before executing
  • RAG retrieval: semantic search via Ollama embeddings + TF-IDF fallback
  • Vision support: paste a screenshot or mockup, get working UI code
  • Task queue: queue multiple prompts for batch processing

IDE

Full-featured editor and workspace

Monaco (the VS Code engine) with syntax highlighting for 15+ languages, fill-in-the-middle code completion, a real terminal, and live preview — all integrated into one window.

  • Monaco editor with multi-cursor, find & replace, minimap
  • FIM autocomplete with any compatible Ollama model
  • Nested file tree with quick-open search (Ctrl+P)
  • Full xterm.js PTY terminal with copy/paste
  • Embedded Vite dev server with run / stop / refresh
  • Package manager: install and remove npm packages from UI
  • Multi-file .env editor
  • Full-text project search (Ctrl+Shift+F)
  • Command palette (Ctrl+K)

Database

SQLite, zero config, built-in browser

No Docker, no PostgreSQL server, no configuration. Deyad uses SQLite — your database is just a file. Prisma ORM handles schema migrations, and the built-in table browser lets you inspect data in real time.

  • SQLite: always works, even fully offline
  • Prisma ORM for type-safe schema management
  • Browse tables, view rows, and inspect schema in-app
  • Agent can query database structure while coding
  • Zero disk overhead vs ~400MB+ for PostgreSQL + Docker

Version Control

Git-first, auto everything

Every AI generation is automatically committed. Browse a visual timeline of commits, preview diffs, and restore any version with one click. Push, pull, and manage branches from the UI or natural language.

  • Auto-commit every agent iteration
  • GitHub / GitLab push, pull, and remote management
  • Create, switch, and list branches from UI or chat
  • Visual commit timeline with one-click restore
  • Diff preview before accepting changes
  • Say "push to github" or "create a feature branch" in chat

Deploy

Ship to 7 targets, one click

Deploy from a single modal to Vercel, Netlify, Railway, Fly.io, Surge, any VPS via SSH, or package as a desktop app with Electron Builder.

  • Vercel & Netlify: OAuth or CLI, one-click deploy
  • Railway & Fly.io: full-stack with database support
  • Surge: instant static site hosting
  • VPS: SSH + rsync with optional nginx + Let's Encrypt SSL
  • Desktop: package as AppImage / .exe / .dmg
  • Mobile preview via Capacitor (Android & iOS)
  • ZIP and PWA export
