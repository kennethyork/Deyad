Getting Started

Get Deyad running in three steps.

1

Install Ollama

Download from ollama.ai and pull a model:

ollama pull llama3.2
2

Install Deyad

Grab the latest release from the downloads page and install it.

3

Create a project

Open Deyad, click New App, choose a stack, and describe what you want to build in the chat panel.

Installation

Ubuntu / Debian

curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash - sudo apt-get install -y nodejs git curl -fsSL https://ollama.ai/install.sh | sh ollama pull llama3.2

Then download Deyad-amd64.deb from GitHub Releases and install:

sudo dpkg -i Deyad-amd64.deb

Windows

Install Node.js, Git, and Ollama. Then download and run Deyad-x64.exe from GitHub Releases.

From source

git clone https://github.com/kennethyork/Deyad.git cd Deyad npm install npm start

First Project

  1. Click New App in the sidebar
  2. Name your project and pick a stack (e.g. Full Stack)
  3. Wait for scaffolding to finish
  4. Open the chat panel and describe what you want
  5. Watch the agent write code, install packages, start the dev server, and fix any errors
  6. Use the live preview panel to see your app running
