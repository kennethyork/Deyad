Getting Started

1. Install Ollama and pull a model: ollama pull llama3.2.

2. Install Deyad and open the app.

3. Create a new project, choose a stack, and tell the chat what to build.

Commands

# Start local dev environment
npm start

# Build release binaries
npm run dist
