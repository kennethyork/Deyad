Getting Started

Follow these three steps to go from zero to a running app in under five minutes.

1

Install prerequisites

Install Node.js 18+, Ollama, and Git. Pull a model:

ollama pull qwen2.5-coder:7b
2

Download & install Deyad

Grab the latest release for your platform from the Download page or build from source:

git clone https://github.com/kennethyork/Deyad.git
cd Deyad
npm install
npm start
3

Create your first project

Click New App, choose a template, and start chatting with the AI. The agent will scaffold your app, install dependencies, and start the dev server automatically.

Installation

Ubuntu / Debian

# Download .deb from GitHub Releases
sudo dpkg -i deyad_*.deb
sudo apt-get install -f

Fedora / RHEL

sudo rpm -i deyad-*.rpm

Windows

# Download and run the .exe installer from GitHub Releases
# Or use winget (when available):
winget install kennethyork.Deyad

Build from source

git clone https://github.com/kennethyork/Deyad.git
cd Deyad
npm install

# Development
npm start

# Production build
npm run make

First Project

  1. Launch Deyad and click New App in the sidebar
  2. Enter a project name and choose a template (or start blank)
  3. A Welcome Wizard walks you through model selection, Git config, and API keys
  4. Open the chat panel and describe what you want to build
  5. Watch the agent scaffold files, install packages, and start the dev server
  6. The live preview updates in real time as the agent edits code
  7. Review changes in the Version History panel, or ask the agent to iterate
