Features

Everything you need in one window

A full IDE, autonomous AI agent, database browser, terminal, and deployment — all running locally.

🤖

Autonomous Agent

18 tools. Up to 30 iterations. Reads your code, writes files, runs commands, detects errors, and self-corrects automatically.

✎

Monaco Editor

The same editor engine as VS Code — syntax highlighting for 15+ languages, keybindings, and AI-powered autocomplete.

👁

Live Preview

Embedded Vite dev server with one-click run, stop, and refresh. See changes instantly as the agent writes code.

🗂

Database Browser

SQLite with Prisma — zero config. Browse tables, inspect schemas, and view data directly inside the app.

🔌

Git Built-in

Auto-commit every generation. Push, pull, branch, and view version history with one-click restore and diff preview.

🌏

Deploy Anywhere

Ship to Vercel, Netlify, Railway, Fly.io, Surge, any VPS via SSH, or package as a desktop app — 7 targets total.

🔩

AI Code Completion

Fill-in-the-middle autocomplete using any FIM-capable Ollama model. Plus RAG search with local embeddings.

🖼

Image to Code

Paste a screenshot or UI mockup into the chat and get working code back. Supports vision-capable Ollama models.

🧩

Plugin Marketplace

Install community plugins for custom tools, agents, and themes. Extend Deyad without touching the core.

Get Started

Up and running in 2 minutes

Install Ollama, download Deyad, and start building.

🐧 Linux (Ubuntu / Debian)

# Install Ollama + pull a model
curl -fsSL https://ollama.com/install.sh | sh
ollama pull llama3.2

# Install Deyad
sudo dpkg -i Deyad-amd64.deb

🏷 Windows

# Install dependencies
winget install Ollama.Ollama
ollama pull llama3.2

# Run the installer
Deyad-x64.exe

📦 Fedora / Arch (RPM / AppImage)

# Fedora
sudo rpm -i Deyad-x86_64.rpm

# Arch / any distro
chmod +x Deyad-x86_64.AppImage
./Deyad-x86_64.AppImage

💻 From Source

git clone https://github.com/theKennethy/Deyad.git
cd Deyad
npm install
npm start
