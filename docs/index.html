CLI Agent

Your terminal. Your AI. Your code.

Like Claude Code — but fully offline, powered by Ollama. An agentic coding assistant that lives in your terminal.

Install

The CLI is bundled in the Deyad repo. Clone, build, and link it globally in three commands:

bash
# Clone & build
$ git clone https://github.com/kennethyork/Deyad.git
$ cd Deyad/cli && npm install && npm run build

# Link globally
$ npm link

# Verify
$ deyad --help

Requirements

  • Node.js 18+ — runtime
  • Ollama — local AI inference
  • Git — version control (for /undo, /diff)

Usage

deyad
# Interactive mode — opens a REPL
$ deyad

# One-shot — describe what you want
$ deyad "add user authentication"

# Pick a model
$ deyad -m qwen3:32b "fix all tests"

# Auto-confirm all changes
$ deyad -y "refactor to TypeScript"

Slash Commands

commands
/help          Show all commands
/model         Switch Ollama model
/add <file>    Add file to context
/drop <file>   Remove from context
/diff          Show all uncommitted changes
/undo          Revert agent's file changes
/run <cmd>     Run a shell command
/compact       Show token & message stats
/clear         Reset conversation
/quit          Exit
