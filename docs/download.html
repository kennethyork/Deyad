Before you start

Prerequisites

Deyad needs a few things installed on your system.

⬢

Node.js ≥ 18

Runs generated apps, Vite dev server, and npm. Install Node.js

🦙

Ollama

Local AI inference engine. Install Ollama, then run ollama pull llama3.2

🔀

Git

Auto-commit, version history, and GitHub sync. Install Git

Alternative

Build from source

git clone https://github.com/kennethyork/Deyad.git cd Deyad npm install npm start

To build distributable binaries: npm run dist
