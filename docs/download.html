Prerequisites

Install these before running Deyad.

Node.js 18+

JavaScript runtime. Download from nodejs.org

Ollama

Local AI runtime. Download from ollama.com

Git

Version control. Download from git-scm.com

Build from source

For contributors or anyone who prefers to compile locally.

git clone https://github.com/kennethyork/Deyad.git
cd Deyad
npm install

# Run in development mode
npm start

# Build distributable packages
npm run make
