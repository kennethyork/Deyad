Quick Start

# Install the CLI
git clone https://github.com/kennethyork/Deyad.git
cd Deyad/deyad-cli/deyad-cli && npm install && node build.mjs
npm link

# Make sure Ollama is running
ollama serve

# Launch interactive mode
deyad

CLI Reference

CommandDescription
deyadInteractive REPL mode
deyad "prompt"One-shot — run a single task and exit
deyad -m model "prompt"Use a specific Ollama model
deyad -y "prompt"Auto-confirm all changes
deyad -p "prompt"Print mode — output result only
deyad --helpShow all flags and options

Slash Commands (REPL)

CommandDescription
/helpShow all commands
/model <name>Switch Ollama model
/add <file>Add file to context
/drop <file>Remove file from context
/diffShow uncommitted changes
/undoRevert agent's file changes
/run <cmd>Run a shell command
/compactShow token & message stats
/clearReset conversation
